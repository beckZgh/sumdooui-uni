# sumdooui-uni

基于 UniApp 开发的偏业务端的移动组件库

## 自定义主题

### 定制单个组件的主题样式
```css
.sd-button {
    --sd-primary-color: blue;
    --sd-button-default-height: 100rpx;
}
```

### 定制全局主题样式

```css
page {
    --sd-primary-color: blue;
    --sd-success-color: green;
    --sd-warning-color: yellow;
    --sd-danger-color : red;
}
```
